
Sistema simples desenvolvido em Python para cadastro e listagem de produtos com persistência de dados em arquivo JSON.

##  Sobre o projeto

Este projeto foi criado com foco em aprendizado, praticando conceitos fundamentais de Python como funções, listas, tratamento de erros e manipulação de arquivos.

Ele permite que os dados dos produtos sejam salvos automaticamente e carregados sempre que o programa é iniciado.

##  Funcionalidades

-  Adicionar produtos com nome e preço
-  Listar produtos cadastrados
-  Salvamento automático em arquivo JSON
-  Carregamento automático dos dados ao iniciar
-  Validação de entrada de preço
-  Limpeza do terminal para melhor organização

##  Tecnologias utilizadas

- Python 3
- JSON
- Módulo `os`

##  Estrutura do projeto

```
SistemaProdutos/
├── main.py
└── produtos.json
```

##  Objetivo

Treinar lógica de programação e fundamentos de desenvolvimento em Python, com foco em organização de código e persistência de dados.
